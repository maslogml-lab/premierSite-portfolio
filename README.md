# Portfolio Personnel — HTML/CSS

Mon tout premier site web, réalisé uniquement en HTML et CSS, sans JavaScript ni base de données.

## Note importante sur la navigation
La barre de navigation ne fonctionne pas en ouvrant simplement le fichier 
dans un navigateur. Elle nécessite une configuration serveur local (WAMP/XAMPP).
Pour consulter chaque section (CV, Projets, Loisirs, Contact), 
ouvrez directement le fichier HTML correspondant depuis les fichiers du dépôt.

## Technologies
- HTML5
- CSS3

## Description
Site portfolio personnel présentant mon profil, mes projets, 
mon CV, mes loisirs et une page de contact.
Première expérience complète de développement web front-end.
